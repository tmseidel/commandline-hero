Commandline-Hero

Rule your commandline tools on every device

Welcome to Commandline-Hero.

This project has the goal to make your command line tools avialable over http via a browser-interface or directly over a JSON interface. The goals are

  • Instead of typing long commands in a prompt you can use a browser-based form to execute a command.
  • You can execute your desktop commandline programs on mobile devices
  • You can use the software as API to This project provides different software
  • A plugin for Eclipse to define commandline calls
  • A generator that generates Java code to execute the described commands
  • A webapplication that can be accessed from a desktop or a mobile device to schedule commands.

A typical scenario for commandline hero is that you want to downscale a video on your smartphone. On a desktop you would use the commandline tool ffmpeg with the following parameter

ffmpeg -i C:\Users\Tom\Videos\MyVideo.mp4 -filter:v scale=400:240 -sws_flags lanczos mySmallVideo.mp4
